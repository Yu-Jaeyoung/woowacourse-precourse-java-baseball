# 숫자 야구 흐름 작성

1. 사용자가 프로그램을 시작
2. 환영 인사
3. 입력 진행
    - 입력한 값을 기반으로 n번 반복해가며 판독
        
        → n번 = 사용자가 3스트라이크 성공할 때까지의 횟수
        
4. n번 반복 진행 후, 게임 종료 문구 출력
5. 게임 새로 시작 또는 종료 여부 문구 출력
    - 새로 시작 선택 시, 3번으로
    - 종료 선택 시, 프로그램 종료
